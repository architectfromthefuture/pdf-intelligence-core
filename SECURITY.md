# Security

## Reporting a vulnerability

Please report security issues **privately** via **GitHub Security Advisories** for this repository when enabled, or to the repository maintainers.

Include:

- Affected component or path
- Safe reproduction steps
- Impact assessment (best effort)

Do **not** open a public issue for undisclosed exploitable bugs.

## Scope

This policy covers this repository and its documented release surfaces. Issues in upstream dependencies should be reported to those projects.
